# LightGBM Models

Déposez ici vos modèles LightGBM exportés via `Booster.save_model`. Le moteur
recherche par défaut un fichier `lightgbm_matcher.txt`. Vous pouvez également
spécifier un autre chemin en définissant la variable d'environnement
`LIGHTGBM_MODEL_PATH` avant de lancer les scripts ou l'API.
